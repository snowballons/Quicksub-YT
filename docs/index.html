How to Use

1

Add Channels

Upload a .csv/.txt file, paste comma-separated channel URLs, or enter a webpage URL to scan for links.

2

Start Processing

Click "Start Processing". The extension will open each channel page one by one.

3

Subscribe Manually

You'll have a 10-second window on each channel page to click the "Subscribe" button yourself.

4

Cooldown & Reset

Respect session limits. After processing a batch or using your session allowance, cooldowns apply. Inactivity resets your session!
